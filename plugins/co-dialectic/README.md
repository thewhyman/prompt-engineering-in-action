## What It Does

One file. Paste it into any AI. Five systems activate automatically:

1. **10 expert personas** — auto-detected, fusible. 🏗️ Architecture (Jeff Dean), 🎨 Design (Jony Ive), 🔍 Debugging (Linus Torvalds), 📦 Product (Shreyas Doshi), 🎯 Positioning (Steve Jobs), 🔗 Career (Reid Hoffman), ⚡ Productivity (Tim Ferriss), 📊 Data (Nate Silver), ✍️ Writing (George Orwell), 🔥 Mindset (Tim Storey).
2. **Per-prompt quality scoring** — every response starts with `📦 Product (Doshi) · 92%`. The tightest feedback loop possible.
3. **Context smoke detector** — 🟢/🟡/🔴 health tracking with auto-handoff before quality degrades.
4. **Auto-codification** — correct once, benefit forever. Every lesson becomes a generative principle.
5. **The AI teaches you back** — names techniques you're using and connects them to broader concepts.

## Architecture (v2.2+)

SKILL.md is the **thin core** — all protocols, works standalone. Extended features (gamification, gifting, commands) live in the repo README's "For Agents" section as a CDN — evolving without reinstall.

See the main [README](../../README.md) for install instructions, screenshots, and the full feature set.
